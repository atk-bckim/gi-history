# AI Copilot Web Guide

> GitHub Copilot을 중심으로 AI 도구 사용법, agentic AI 개념, Microsoft Copilot 업무 활용, 고난도 AI 웹디자인 쇼케이스를 교육하는 한/영 웹 가이드.
> Next.js App Router, TypeScript, MDX, Tailwind CSS, Framer Motion, React Three Fiber 기반으로 문서형 학습과 인터랙티브 실습을 함께 제공한다.

---

## 목차

### 현재 상태
- MVP 앱은 `http://localhost:3000`에서 실행 가능하다.
- 3D Digital Twin, Semiconductor Explorer, Agentic Workflow Control Room 쇼케이스는 모두 인터랙티브 페이지로 제공된다.
- GitHub Copilot 개요, VS Code, CLI, cloud/coding agent, review/security/admin, Microsoft Copilot, Agentic AI까지 7개 핵심 학습 모듈을 한/영 MDX로 제공한다.
- Copilot VS Code, CLI, cloud/coding agent, review 과정은 표면 선택 매트릭스, 컨텍스트 팩, 승인 매트릭스, 이슈 브리프, PR 리뷰 계약, 정책 매트릭스, 보안 triage runbook까지 포함한다.
- Agentic AI 과정은 딥 인터뷰 스크립트, 브레인스토밍 퍼널, 서브에이전트 위임 브리프, 하네스 준비도 스코어카드, Ralph Wiggum Loop 방지책까지 포함한다.
- Microsoft Copilot 과정은 부서별 rollout playbook, 회의 요약 프롬프트, connector 결정 등록부, Purview 증거 체크리스트, agent knowledge-source 검토까지 포함한다.
- 역할별 학습 경로는 개발자, 기술 리더, 업무 사용자, 교육자별 lesson 순서와 산출물, 완료 증거를 제공한다.
- Copilot Workflow Simulator는 checkout 중복 제출 사고를 issue brief, context pack, agent-mode dry run, no-write investigation, approval gate, verification evidence, PR review contract까지 이어지는 end-to-end 실습으로 확장했다.
- lesson 목차는 active section과 읽기 진행률을 제공하며, header/locale/labs/quiz/showcase keyboard 접근성 회귀 테스트가 있다.
- 용어집은 29개 핵심 개념으로 확장했고, 자료실에는 한/영 정적 검색을 제공한다.
- 현재 구현은 정적 교육 사이트이며, 외부 AI API/로그인/DB 저장/실제 산업 데이터 연동은 제외되어 있다.
- 최신 검증: unit 45개, e2e 49개, lint, typecheck, build 통과.
